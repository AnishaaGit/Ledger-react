# Personal Ledger App

A simple personal finance tracker built using **React**, **Zustand**, **Tailwind CSS**, and **Recharts**.

## Features
- Add income and expense transactions  
- View total income, expense, and balance  
- Pie chart visualization (income vs expense)  
- Transaction history with delete option  
- Responsive dark-themed UI  

## Tech Stack
- React  
- Zustand  
- Tailwind CSS (v4)  
- Recharts
